# Dev Backend - StartupSeal

Essential backend services for StartupSeal platform.

## Structure
- `src/services/` - Core services (Walrus, Nautilus, AI)
- `src/routes/` - API routes
- `ai/` - Python ML trust scorer
- `src/server.js` - Main Express server

## Setup
```bash
npm install
cd ai
pip install -r requirements.txt
```

## Run
```bash
npm start
```
